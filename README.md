# Dashboard de Análisis de Vehículos

## Descripción del Proyecto

Esta aplicación web interactiva permite realizar un análisis exploratorio de datos de anuncios de venta de vehículos usados. La aplicación está construida con **Streamlit** y **Plotly Express** para proporcionar visualizaciones interactivas y fáciles de usar.

## Funcionalidades

La aplicación web proporciona las siguientes características:

- **Histograma interactivo**: Visualiza la distribución del kilometraje de los vehículos
- **Streamlit**: Framework para crear la aplicación web
- **Plotly Express**: Librería para crear visualizaciones interactivas
- **Pandas**: Manipulación y análisis de datos

## Cómo Ejecutar la Aplicación

1. Clona este repositorio
2. Instala las dependencias: `pip install -r requirements.txt`
3. Ejecuta la aplicación: `streamlit run app.py`
4. Abre tu navegador en `http://localhost:8501`
